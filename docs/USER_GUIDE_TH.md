# คู่มือใช้งาน (ภาษาไทย)

## พนักงาน
1. เข้าสู่ระบบด้วยอีเมล + เลือก role `employee`.
2. เปิด `Employee Dashboard` เพื่อตรวจยอดประจำวัน.
3. บันทึกรายการในเมนู: ยอดขาย, ค่าใช้จ่าย, ฝากเงิน, ลูกหนี้, ยกเลิกบิล, กระทบยอดเงินสด.
4. ตรวจผลต่างเงินสดทุกวันก่อนปิดรอบ.

## เจ้าของ
1. เข้าสู่ระบบด้วย role `owner`.
2. ดู `Owner Dashboard` เพื่อสรุปยอดวันนี้/สัปดาห์/เดือน.
3. เข้าเมนูอนุมัติยกเลิกบิล และตัดสินใจ `Approve/Reject`.
4. ตรวจ `Audit Log` เพื่อดูว่าใครบันทึกอะไร เวลาใด.

## หลักการควบคุมสำคัญ
- ห้ามแก้ไข/ลบข้อมูลโดยตรงในชีต
- ใช้ระบบเป็นจุดบันทึกเดียว (single source of truth)
- ต้องมีหลักฐานแนบสำหรับรายการสำคัญ
